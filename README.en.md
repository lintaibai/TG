## TG OpenSourance contains the following parts

- NexusApi (Tiangong Node backend)
- NexusReact (Tiangong React frontend)
- NexusVue (Tiangong Vue3 frontend)
- NexusApp (Tiangong app end --- currently being written
- NexusHarmony (Tiangong Harmony end --- currently being written)
- NexusNote (Tiangong document --- has been written and entered the personal blog)
- NexusJava (Tiangong Java end --- currently being planned)

## Project running and environment dependencies

### (Database environment)

Mysql 8.3.0

### (Backend NexusApi environment)

Node v20.12.0

### (Frontend environment)

React v18.2.0
